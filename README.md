# Golden Crumbs Cake Shop Admin Portal

## Project Overview

Golden Crumbs Cake Shop Admin Portal is a React Single Page Application (SPA) created to help administrators manage cake products for an e-commerce cake shop. The application allows users to add, view, update, delete, and search for cake products using a simulated backend database.
The project was developed using modern frontend technologies and follows the Software Development Life Cycle (SDLC) process.
# SDLC Process
## 1. Planning
The goal of the project was to build a responsive admin dashboard for managing cake products in an e-commerce environment.
Main requirements included:
- Landing page
- Product management
- CRUD operations
- Search functionality
- Client-side routing
- Responsive design
- Database persistence
- Testing
## 2. Analysis
The application needed to:
- Store cake product information
- Allow administrators to manage products
- Provide smooth navigation between pages
- Update data dynamically without reloading the page
Users can:
- Add new cakes
- View all cakes
- Edit cake information
- Delete cakes
- Search for cakes
## 3. Design
The application was designed as a Single Page Application using React Router for navigation.
Pages included:
- Home Page
- Products Page
- Add Cake Page
- Edit Cake Page

The interface uses:
- Gold
- Blue
- White
- Black
The design is responsive for desktop and mobile devices
## 4. Development
The project was developed using:
- React
- Vite
- Tailwind CSS
- React Router DOM
- Axios
- JSON Server

Features implemented:
- CRUD operations
- Dynamic search
- Custom hooks
- Responsive layouts
- API requests using Axios
- Routing between pages
A `db.json` file was used to simulate a backend database.
## 5. Testing
Testing was completed using:
- Vitest
- React Testing Library

Tests were created to verify:
- Navigation rendering
- Component functionality
- User interactions
## 6. Deployment and Maintenance
The project was uploaded to GitHub for version control and deployed using Vercel.
Git branches were used during development to manage features separately before merging into the main branch.

# How to Run the Project
## 1. Clone the Repository
```bash
git clone YOUR_GITHUB_LINK
```
## 2. Open the Project Folder
```bash
cd golden-cakes
```
## 3. Install Dependencies
```bash
npm install
```
## 4. Start the React Application

```bash
npm run dev
```
## 5. Start JSON Server
Open a second terminal and run:
```bash
npm run server
```
# Database
The project uses a `db.json` file to store cake product data locally using JSON Server.
